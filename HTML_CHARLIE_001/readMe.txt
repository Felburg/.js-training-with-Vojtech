Purpose of the project is to make:
  1. The pop-up box with a text which will be defined by a user.
  2. The pop-up box will be shown as many times as user defines it.
  3. The pop-up box will start showing when user press a button.
  
Components:  
  1. Two textfields with ability for user to enter text and numbers in them.
  2. First textfield will be populated by text. Text is defined by users input.
  3. Second textfield will be populated by a number. Number is defned by uses input.
  4. Button will trigger showing the pop-up window.
  
Used technologies:
  1. HTML
  2. CSS
  3. JavaScript
